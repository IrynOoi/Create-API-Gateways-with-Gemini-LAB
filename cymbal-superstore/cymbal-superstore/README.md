Cymbal Superstore is sample application based on a large grocery retailer, built with React.js, Express.js (Node), and Google Cloud Firestore.

This sample is used to showcase [Duet AI in Google Cloud](https://cloud.google.com/duet-ai), including:

- Code Completion
- Code Generation
- Code Chat
- Console Chat
- Console-based SQL code generation (BigQuery, Cloud Spanner)
- AI-generated summaries (Security Command Center, Cloud Logging, BigQuery)

**[Check out the blog post to learn more.](https://cloud.google.com/blog/products/application-development/how-ai-driven-software-creation-tools-speed-up-your-development)** (More content coming soon.)

This is a demo, with mocked components. It is not an official Google product.
